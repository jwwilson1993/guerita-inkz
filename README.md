# Guerita Inkz

A modern website for a tattoo-focused brand, built with Next.js, TypeScript, and responsive front-end design.

Guerita Inkz is a front-end project focused on creating a polished web presence for a tattoo brand. The project emphasizes visual design, responsive layouts, and a clean user experience across desktop and mobile.
